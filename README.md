# 原稿を簡単に書くためのテンプレートフォルダ

        このリポジトリは原稿を書くためのツールなどを集めたものになる予定です

## 依存関係

このリポジトリは次のツールを使います。

- GNU Make
- シェルスクリプト
- Python 3.\*

## できること

- 実装済み
    - Markdown で書いた原稿を pandoc で HTML に変換する。
    - Make 一発で最新の md ファイルを pandoc で HTML に変換する。

- やりたいこと
    - Make 一発で pdf にする。

## 内容物

- Makefile
- CSS

などを予定しています。
